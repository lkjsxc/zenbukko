# Typography

## Font Stacks

- UI: `system-ui, -apple-system, Segoe UI, sans-serif`
- Code: `ui-monospace, SFMono-Regular, Menlo, monospace`

## Scale

| Name | Size |
|------|------|
| xs | 12px |
| sm | 14px |
| base | 16px |
| lg | 20px |
| xl | 24px |

## Usage

Headings use lg/xl. Form labels sm with semibold. Log panel uses code stack at sm.
